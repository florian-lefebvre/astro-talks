# Better API to retrieve <span class="text-gradient font-normal">font buffers</span>

- Currently only works in development and during SSR
- Have a unified API that works during **prerendering**

```ts
import { fontData, experimental_getFontBuffer } from "astro:assets";

// Dev & prerendering
const fontBuffer = await experimental_getFontBuffer(
  fontData["--font-inter"][0].src[0].url,
);

// SSR
const fontBuffer = await experimental_getFontBuffer(
  fontData["--font-inter"][0].src[0].url,
  context.url,
);
```
