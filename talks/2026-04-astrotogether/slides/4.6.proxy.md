# <span class="text-gradient font-normal">Proxy</span> provider URLs

For self hosting, we need control over URLs

```ts
for (const source of font.src) {
  // We handle protocol relative URLs here so they're not considered absolute FS paths
  const originalUrl = source.url.startsWith("//")
    ? `https:${source.url}`
    : source.url;

  // ttf => truetype
  const format = getFormat(source);

  const url = getProxyUrl({ family, source, originalUrl, format });

  source.originalURL = originalUrl;
  source.url = url;
  source.format = format;
}
```

- Original: `https://fonts.gstatic.com/s/roboto/v51/KFO5CnqEu92Fr1Mu53ZEC9.woff2`
- Proxied: `/_astro/fonts/<hash>.woff2`
